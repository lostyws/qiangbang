1.一对一模式：雇佣服务商、购买服务
在帮抢网您可以找到优秀的服务商解决您的需求，您可以查看服务商的店铺，然后选择直接购买服务或是雇佣服务商为您工作。
交易流程：雇佣服务商/ 购买服务 —>确认赏金和细节 —> 托管赏金 —> 服务商开始工作 —> 验收合格 —> 付款

2.悬赏模式： 比稿、招标、计件
如果您的需求比较复杂或者需要多个服务商完成，您还可以直接发布需求，众多服务商参与，使您的需求得到圆满解决。
交易流程：发布需求（比稿/招标/计件） —>托管赏金 —>服务商投稿 —>选标 —>付款

3.悬赏模式详解
招标： 适用于大多数需求，特别 适合建筑设计、网站开发、软件开发、装修设计等周期较长、工作较复杂的项目类需求，需要找人单独服务。 发布需求后，服务商会对您的需求作出报价，您选择一个最满意的服务商进行一对一服务。


比稿： 适合 Logo设计、标志、海报、名片设计、起名取名等需要很多创意的需求。 托管赏金后，服务商将提供实质性的稿件，您可从众多稿件中，选择最满意的，验收并付款。

计件： 适合微博、论坛、QQ群推广、市场调查、短信祝福类需求，需要众多人重复做一件简单的事情。 托管赏金后，服务商完成任务并交稿（上传证据），您验收工作，合格一个支付一个，余款100%退回。

1. 发布需求出来，如果有服务商来交稿，我怎么可以跟服务商联系沟通？
您可以下载咱们的APP，随时与服务商沟通。 使用APP，您可以直接在稿件下评论；也可以选择服务商中标，获得服务商电话，然后跟服务商电 话联系修改；如果对方是签约服 务商，您还可以随时与服务商沟通。

2.  需求发布出来之后，什么时候可以收到服务商的交稿？我怎么可以知道有服务商来交稿的？
咱们网站默认的需求的投标征集期是 7 天，您在发布需求成功的 7 天内都可能收到来自服务商的交稿，建议您下载APP随时关注您的需求交稿情况。
 
3. 需求发布出来，是不是选择中标之后（雇佣服务商托管赏金之后）钱就马上支付给服务商呢？
不是。选择中标只是确定您和服务商的合作关系，使您可以直接跟服务商沟通合作项目。您的赏金是托管到帮抢网平台，只有您验收工作合格后网站才会把钱付款给服务商，保障双方诚信交易。
 
4.是不是我一直不验收付款，钱就一直在帮抢平台上面的？
不一定。正常的情况下如果您一直不验收付款，网站就不会付款给服务商。但如果服务商已经完成工作，可以发起提醒付款申请，您需要在申请发起的 7 天内进行操作，选择同意或暂不付款。如果您未进行操作， 7 天后钱会自动付款给服务商。如您对服务商完成的工作有异议，您可以向网站申请维权。
 
5.交易进行时如果服务商没有能力完成我的工作，是不是钱就一定要支付给服务商？
不是。您可以跟服务商协商交易的处理；如果协商不成功，您还可以拨打帮抢网诚信服务专线，由帮抢网协调处理。
